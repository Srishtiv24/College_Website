# College_Website

This project is a simple and elegant college website built using HTML, CSS , Bootstrap & JS . The focus is on creating a clean user interface with well-structured pages that showcase essential sections like the homepage, about, courses, and contact details. The design emphasizes readability, responsive layout, and a professional look suitable for an academic institution.
